# Custom rice

Monochrome rice based on openSUSE Tumbleweed & the niri compositor

## SysInfo

- Distro: OpenSUSE Tumbleweed x86_64
- WM: niri
- shell: fish
- terminal: kitty
- bar: waybar
- launch: fuzzel
- notification: mako
